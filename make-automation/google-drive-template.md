# Google Drive Template

## Purpose

Use Google Drive as the central file library for brand assets, generated images, exported captions, scripts, and approval files.

## Recommended folder structure

```text
PM Digital Content Studio/
  Vaughan Mortgage Group/
  eStack/
    Logos/
    Affiliate Assets/
    Blog Images/
    Social Posts/
    JSON Exports/
  The Straight Cut/
  Cut and Hinge/
```

## Make.com actions

1. Search files by brand and asset ID.
2. Get share link.
3. Write link back to the `Image Library` or `Content Library` tab.
4. Use file link in content approvals or publishing packages.

## Naming convention

```text
BRAND_PLATFORM_DATE_TOPIC_VERSION
```

Example:

```text
eStack_Instagram_2026-07-10_AffiliateDeals_v1.png
```
